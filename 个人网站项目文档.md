## 1. 项目愿景与目标

### 1.1 一句话描述

> 一个让访客在 **30 秒内记住你**、在 **3 分钟内了解你的核心能力**、并愿意 **点击联系或查看 GitHub** 的极客风个人网站。

### 1.2 核心目标

| 目标 | 说明 | 衡量指标 |
|------|------|----------|
| **展示技术实力** | 网站本身就是你的作品集 — 动效、3D、终端交互即能力证明 | 访客停留 > 2 分钟 |
| **建立个人品牌** | 统一的视觉语言 + 清晰的技术标签 + 个人叙事 | 社交媒体分享率 |
| **降低沟通成本** | 招聘方 / 合作方快速获取：你是谁、能做什么、做过什么 | 联系表单转化率 |
| **可维护可扩展** | 内容数据驱动，新增项目 / 博客无需改核心代码 | 更新一篇内容 < 15 分钟 |

### 1.3 设计哲学（借鉴顶级案例的共识）

1. **网站即作品**：不要只「描述」你会什么，要「演示」你会什么（Robby Leonardi、Bruno Simon 的核心理念）。
2. **滚动即叙事**：把简历变成一条 journey — 从 Boot 到 Contact，像通关一样浏览（Robby Leonardi 关卡式结构）。
3. **沉浸但不迷失**：炫酷特效服务于信息传递，每个 Section 有明确 CTA 和阅读路径。
4. **双模式体验**：参考 gitvictoralves/portfolio — 提供「极客沉浸模式」和「简洁阅读模式（Recruiter Mode）」，兼顾 HR 与同行。
5. **性能是设计的一部分**：再炫的 3D，移动端卡顿 = 负分。

## 2. 优秀案例调研与灵感来源

### 2.1 经典标杆案例

#### 🎮 Robby Leonardi — 互动游戏式简历
- **网址**：https://rleonardi.com/interactive-resume/
- **亮点**：
  - 横版卷轴 + 角色闯关，滚动 / 方向键控制前进
  - 每个「关卡」对应一段经历：About → Skills（水下世界）→ Experience（图表可视化）→ Awards → Contact
  - HTML5 Canvas + GSAP 动画时间线 + CSS Transform
- **可借鉴**：
  - **关卡式信息分区**：把你的职业路径拆成 4–6 个「Zone」
  - **进度感**：顶部或侧边显示 `Level 1/5 · Skills Unlocked`
  - **彩蛋**：隐藏成就（Konami Code、输入 secret command 等）

#### 🚗 Bruno Simon — 3D 可驾驶作品集
- **网址**：https://bruno-simon.com/
- **亮点**：
  - Three.js + Cannon.js 物理引擎，遥控车探索 3D 世界
  - 零传统 UI，信息嵌入 3D 场景（项目牌、社交链接变成场景物体）
  - Awwwards SOTD / FWA FOTY 获奖
- **可借鉴**：
  - 3D 作为 **导航媒介** 而非纯装饰
  - 移动端降级策略（降低粒子数、关闭物理、简化模型）
  - 开源参考：`brunosimon/folio-2019`、`folio-2025`

#### 🖥️ Cyberpunk Terminal 系列
- **代表**：Anisul Islam Terminal Portfolio、Arun Kushwaha One Page Portfolio、clearPath-Dev/terminal-portfolio
- **亮点**：
  - 全屏终端命令行：`help`、`ls`、`cat about.txt`、`open projects/`
  - CRT 扫描线、RGB Glitch、Matrix 雨、开机 Boot Sequence
  - 命令即导航，极客身份 instantly 建立
- **可借鉴**：
  - Hero 区嵌入可交互 Terminal CLI
  - 命令历史、Tab 补全、主题切换命令（`theme neon`、`theme matrix`）

#### ✨ Awwwards 获奖现代开发者作品集
- **代表**：Mohit Virli、Federico Pian、Romain Vincens、gitvictoralves/portfolio
- **共性**：
  - GSAP ScrollTrigger 电影感滚动
  - Glassmorphism 玻璃卡片 + 语义化霓虹光晕
  - 主题切换（深色为主，可选浅色 Recruiter Mode）
  - Lenis 平滑滚动 + Framer Motion 微交互

### 2.2 风格光谱定位

你的目标风格在光谱中的位置：

```
纯终端 CLI ──────●────────────────── 纯 3D 游戏世界
     (Matrix)    你的定位              (Bruno Simon)
              极客科技风
         终端 + 霓虹 + 粒子 + 轻量 3D
              + 滚动闯关叙事
```

**推荐策略**：以 **2D 滚动叙事 + 终端交互** 为主干（开发成本可控），Hero / About 区加入 **轻量 3D 粒子场景**（非全站 3D，避免 Bruno 级别的工作量）。

### 2.3 竞品差异化建议

| 常见做法 | 你的差异化 |
|----------|------------|
| 静态卡片堆叠 | 滚动驱动「代码编译进度条」式经历展示 |
| 假终端（仅装饰） | **真可交互 CLI**，能 `git log --oneline` 看项目 |
| 全屏粒子拖慢性能 | 粒子随滚动变形（Codrops GSAP + WebGL 教程思路） |
| 只有深色模式 | 一键切换 Recruiter 简洁模式（白底、无动画） |

## 4. 视觉设计系统

### 4.1 色彩系统

#### 主色板（Cyberpunk Neon）

| Token | 色值 | 用途 |
|-------|------|------|
| `--bg-primary` | `#050508` | 主背景（近黑，略偏蓝） |
| `--bg-secondary` | `#0a0a12` | 卡片 / Section 背景 |
| `--bg-elevated` | `#12121f` | 浮层、Modal |
| `--neon-cyan` | `#00f0ff` | 主强调色、链接、光标 |
| `--neon-magenta` | `#ff00aa` | 次强调、Hover、Glitch 层 |
| `--neon-green` | `#39ff14` | 终端文字、成功状态 |
| `--neon-purple` | `#a855f7` | 渐变辅助、Skill 标签 |
| `--neon-amber` | `#ffb700` | 警告、高亮数据 |
| `--text-primary` | `#e8e8f0` | 正文 |
| `--text-muted` | `rgba(232,232,240,0.55)` | 次要文字 |
| `--border-glow` | `rgba(0,240,255,0.25)` | 卡片边框光晕 |

#### 渐变预设

```css
/* 主标题渐变 */
--gradient-hero: linear-gradient(135deg, #00f0ff 0%, #a855f7 50%, #ff00aa 100%);

/* 卡片背景 */
--gradient-card: linear-gradient(145deg, rgba(0,240,255,0.06) 0%, rgba(168,85,247,0.04) 100%);

/* 按钮 Hover */
--gradient-btn: linear-gradient(90deg, #00f0ff, #a855f7);
```

#### 语义色

| 状态 | 色值 |
|------|------|
| Success | `#39ff14` |
| Error | `#ff3860` |
| Warning | `#ffb700` |
| Info | `#00f0ff` |

### 4.2 字体系统

| 用途 | 推荐字体 | 备选 |
|------|----------|------|
| **代码 / 终端** | JetBrains Mono | Fira Code, IBM Plex Mono |
| **英文标题** | Orbitron / Space Grotesk | Exo 2, Rajdhani |
| **中文正文** | 思源黑体 / 阿里巴巴普惠体 | 霞鹜文楷（如需 softer 感） |
| **英文正文** | Inter / Geist Sans | DM Sans |

```css
/* 字体加载示例 */
--font-mono: 'JetBrains Mono', 'Fira Code', monospace;
--font-display: 'Orbitron', 'Space Grotesk', sans-serif;
--font-body: 'Inter', 'PingFang SC', 'Microsoft YaHei', sans-serif;
```

**排版比例**：
- Hero 主标题：`clamp(2.5rem, 6vw, 5rem)` + `letter-spacing: 0.05em`
- Section 标题：`clamp(1.5rem, 3vw, 2.25rem)`
- 正文：`16px / 1.7`（移动端 `15px`）
- 终端文字：`14px`，`line-height: 1.5`

### 4.3 间距与布局

- **最大内容宽度**：`1200px`（正文区 `720px` 用于 Blog）
- **Section 垂直间距**：桌面 `120px`，移动 `80px`
- **网格**：12 列，gutter `24px`
- **圆角**：卡片 `12px`，按钮 `8px`，终端窗口 `4px`（偏硬朗）

### 4.4 核心视觉特效

| 特效 | 描述 | 强度建议 |
|------|------|----------|
| **Neon Glow** | `box-shadow: 0 0 20px var(--neon-cyan)` | 中等，避免全页发光 |
| **CRT Scanlines** | 半透明横线 overlay + 偶尔 sweep 闪光 | Hero + Terminal 区 |
| **Glitch Text** | RGB 分离 + clip-path 抖动 | 仅 Hero 姓名，2–3 秒循环 |
| **Glassmorphism** | `backdrop-filter: blur(12px)` + 半透明边框 | 卡片、Nav、Modal |
| **Matrix Rain** | Canvas 字符雨 | 背景层，低 opacity（0.08–0.15） |
| **Particle Field** | WebGL Points，1500–2500 粒子 | Hero 背景，随鼠标微动 |
| **Grid Floor** | CSS 透视网格 / Three.js GridHelper | About 或 Skills 区 |
| **Film Grain** | 静态噪点 SVG overlay | 全页 `opacity: 0.04` |
| **Custom Cursor** | 十字准星 / 发光圆点 + trailing | 桌面端，移动端禁用 |

### 4.5 组件设计规范

#### 导航栏
- 固定顶部，滚动后背景 blur + 底部 neon 线
- Logo：`{` + 姓名缩写 + `}`  monospace 风格
- 链接 Hover：下划线从左到右 neon 扫过
- 右侧：`[ Recruiter Mode ]` 切换按钮

#### 按钮
```
Primary：渐变边框 + 透明底 + Hover 填充
Secondary：ghost 样式，neon 边框

#### 卡片（项目 / 技能）
- 玻璃背景 + 左上角切角（corner-cut，参考 NeonBlade UI）
- Hover：轻微 3D tilt（vanilla-tilt.js 或 CSS `perspective`）
- 边框：1px solid + 外发光

#### 终端窗口
```

│ ■ ■ ■                                          │

│ $ whoami                                       │

## 6. 各模块内容规划

### 6.1 Hero / Boot Sequence

**首屏目标**：3 秒内传达身份，制造「哇」的第一印象。

**内容结构**：
```
[Boot 动画序列 — 可选，3–5 秒，可 Skip]
> KERNEL: portfolio-os v1.0.0 booting...
> INIT: particle_engine ✓
> INIT: neon_renderer ✓
> WELCOME, [YOUR_NAME]

[主视觉]
—— 大标题（Glitch + 渐变）——
[你的姓名]

—— 副标题 ——
> Full-Stack Developer · Open Source Enthusiast

—— 终端输入框（可交互）——
$ Enter command or scroll to explore_
[Tab 补全: help | about | projects | contact]

—— CTA ——
[ View Projects ]  [ Download Resume ]  [ GitHub → ]
```

**动效**：
- 背景：WebGL 粒子场 + 缓慢旋转线框几何体
- 文字：逐字 typewriter 效果
- 滚动提示：底部 `↓ SCROLL TO INIT` 闪烁

### 6.2 About — 关于我

**布局**：左图右文 或 左 3D 头像 + 右侧 Markdown 风排版

**内容模板**：
```markdown
## // about.me

你好，我是 [姓名]。

[2–3 句话个人介绍：背景 + 热情 + 当前方向]

- 📍 所在地：[城市]
- 💼 当前：[公司/学校/自由职业]
- 🎯 专注：[前端 / 后端 / 全栈 / AI / DevOps ...]
- ☕  fun fact：[例：深夜 Debug 时只喝冰美式]

### 数字概览（可选）
| 指标 | 数值 |
|------|------|
| 工作年限 | X 年 |
| 开源项目 | X 个 |
| GitHub Stars | X+ |
| 技术文章 | X 篇 |
```

**视觉**：个人照片加 rotating neon ring；或使用 3D 低多边形头像（Three.js）。

### 6.3 Skills — 技能树

**展示形式（三选一或组合）**：

1. **分类标签云**：Frontend / Backend / DevOps / Tools，neon badge
2. **雷达图 / 六边形能力图**：Canvas 或 SVG 动画绘制
3. **RPG 技能树**：节点解锁式，Hover 显示熟练度 `████████░░ 80%`

**数据结构**：
```typescript

  level: number;          // 1-5 或 0-100

  highlight?: boolean;    // 核心技能
}

const skills = {
  frontend: ['React', 'TypeScript', 'Next.js', 'Tailwind CSS', 'Three.js'],
  backend:  ['Node.js', 'Python', 'PostgreSQL', 'Redis'],
  devops:   ['Docker', 'GitHub Actions', 'AWS', 'Linux'],
  tools:    ['Git', 'Figma', 'VS Code', 'Obsidian'],
};
```

### 6.5 Projects — 项目展示

**布局**：Bento Grid（不规则网格）或 3 列卡片

**每个项目卡片包含**：
- 封面图 / 动态预览（GIF 或 hover video）
- 项目名 + 一句话描述
- Tech Stack tags
- Links：Live Demo · GitHub · Case Study
- 数据指标（如有）：`↑ 10k users · ⭐ 500 stars`

**详情页（/projects/[slug]）结构**：
```
1. Hero — 项目封面 + 标题
2. Overview — 背景、目标、你的角色
3. Tech Stack — 架构图
4. Challenges — 技术难点与解决方案
5. Results — 数据成果、截图
6. Links — Demo / Repo
7. Navigation — Prev / Next Project
```

**数据模板**：
```typescript
interface Project {
  slug: string;
  title: string;
  tagline: string;
  description: string;
  cover: string;
  tags: string[];
  links: {
    live?: string;
    github?: string;
    demo?: string;
  };
  featured: boolean;       // 是否在首页突出
  year: number;
  metrics?: Record<string, string>;
}
```

### 6.6 Blog — 技术博客（可选）

**首屏**：最新 3 篇文章卡片 + `View All →`

**文章页**：代码高亮（Shiki / Prism）+ 深色主题 + 阅读进度条

## 7. 交互与动效设计

### 7.1 滚动动画系统

**推荐技术栈**：GSAP ScrollTrigger + Lenis Smooth Scroll

| 动画类型 | 应用场景 | 实现方式 |
|----------|----------|----------|
| **Fade Up + Stagger** | 卡片列表进入 | `y: 60 → 0, opacity: 0 → 1, stagger: 0.1` |
| **Scrub Parallax** | 背景粒子 / 文字 | ScrollTrigger `scrub: true` |
| **Pin + Timeline** | Skills 六边形绘制 | Section pin 住，滚动驱动进度 |
| **Horizontal Scroll** | Projects 横向浏览 | GSAP horizontal scroll section |
| **Scale Down** | Section 离开时 | Apple 风格 sticky scale（参考 ryanmccann1024/portfolio） |
| **Character Reveal** | 大标题 | 逐字 `clip-path` 或 split text |
| **Progress Bar** | 关卡进度 | 滚动百分比映射到 neon 进度条 |

### 7.2 粒子 / 3D 背景

**Hero 粒子场景规格**：
```
- 引擎：Three.js + @react-three/fiber + drei
- 粒子数：桌面 2000 / 移动 500
- 交互：鼠标移动 → 相机轻微偏移（±5°）
- 滚动：粒子随 scroll 扩散 / 收缩
- 几何体：2–3 个 wireframe 多面体缓慢旋转
- 降级：prefers-reduced-motion → 静态渐变背景
```

**性能预算**：
- 3D 场景 JS bundle lazy load
- `dpr` clamp 至 `[1, 2]`
- 移动端检测 → 禁用 3D，改用 CSS 渐变 + 轻量 Canvas 粒子

### 7.3 终端 CLI 交互

**支持命令规划**：

| 命令 | 行为 |
|------|------|
| `help` | 列出所有可用命令 |
| `whoami` | 显示个人信息 |
| `ls` | 列出「目录」：about, projects, skills, contact |
| `cat about.txt` | 输出 About 内容 |
| `cd projects` | 滚动到 Projects 区 |
| `open [url]` | 新窗口打开链接 |
| `theme [name]` | 切换 neon / matrix / recruiter |
| `sudo hire-me` | 彩蛋：显示「Permission granted」+ 联系方式 |
| `clear` | 清空终端 |
| `exit` | 关闭终端 overlay |

**技术实现**：
- 命令解析器：自写 tokenizer 或 `commander` 风格
- 历史记录：↑↓ 键浏览
- 输出动画：逐行 typewriter

### 7.4 微交互清单

- [ ] 按钮 Hover：magnetic effect（向光标靠近）
- [ ] 链接 Hover：neon underline sweep
- [ ] 卡片 Hover：3D tilt + glow intensify
- [ ] 页面切换：glitch transition（如有子页）
- [ ] 加载：Boot sequence 或 skeleton shimmer
- [ ] 复制邮箱：点击 → `Copied!` toast
- [ ] Konami Code：触发 Matrix 模式
- [ ] `Cmd+K` / `Ctrl+K`：Command Palette 快速导航

### 7.5 Recruiter Mode（简洁模式）

**触发**：Nav 按钮 / 终端命令 `mode recruiter`

**变化**：
- 关闭所有动画和 3D
- 切换为白底 / 深灰文字
- 线性布局，无粒子
- 保留完整信息和时间线
- 顶部 Banner：`📄 Optimized for quick reading`

## 9. 性能、无障碍与 SEO

### 9.1 性能预算

| 指标 | 目标 |
|------|------|
| LCP | < 2.5s |
| FID / INP | < 100ms |
| CLS | < 0.1 |
| 首屏 JS | < 200KB gzip（不含 3D lazy chunk） |
| 3D 场景加载 | lazy on viewport / user interaction |

**优化手段**：
- `next/image` 自动优化图片（WebP/AVIF）
- 3D 组件 `dynamic(() => import(...), { ssr: false })`
- 字体 `next/font` subset 加载
- GSAP / Three.js tree-shaking
- 移动端降级策略

### 9.2 无障碍（a11y）

- `prefers-reduced-motion`：禁用动画和 3D
- 终端 CLI 不影响 Screen Reader（提供 skip link）
- 所有交互元素可键盘导航（Tab / Enter / Esc）
- 颜色对比度：正文 ≥ 4.5:1
- 图片 `alt` 文本
- Recruiter Mode 作为 a11y 友好备选

### 9.3 SEO

```typescript
// metadata 示例
export const metadata = {
  title: '你的姓名 | Full-Stack Developer',
  description: '个人简介 150 字以内，包含核心关键词',
  keywords: ['前端开发', 'React', 'TypeScript', '个人网站'],
  openGraph: {
    title: '...',
    description: '...',
    images: ['/og-image.png'],  // 1200x630 neon 风格
  },
};
```

**必备**：
- `sitemap.xml` + `robots.txt`
- JSON-LD 结构化数据（Person + WebSite）
- OG Image 自动生成（可选 `@vercel/og`）
